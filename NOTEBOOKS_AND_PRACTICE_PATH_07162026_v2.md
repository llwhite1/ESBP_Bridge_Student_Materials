## Suggested use

1. Open the current day workbook and complete its readiness reasoning.
2. Use the workbook's mapped activity IDs to choose the matching VIP notebook.
3. Work in the provided sample and transfer cells without deleting or recreating them.
4. Save one completed `.ipynb` and submit that file to the matching Gradescope assignment.

The current notebooks do not use `%%writefile`, generated ZIP files, or hidden function wrappers. If a notebook asks for those steps, it is an older superseded copy.
