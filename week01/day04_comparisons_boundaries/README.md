## What you are learning today

- Focus: Comparisons and boundaries.
- Main evidence to keep: boundary values, comparison results, threshold explanations.
- Why it matters: Reason about exact thresholds, boundary cases, and comparison outcomes.

## Before class / during class / after class

- Before class: Identify every threshold or cutoff before solving.
- During class: Test values just below, at, and above the boundary.
- After class: Write one boundary case that protects your answer.

## If you get stuck

If stuck, compare one concrete value against one condition at a time.

Try to name the kind of evidence you need before asking for help: output, state, type, branch decision, test, loop pass, list state, or support plan.
