## What you are learning

- Target: Connect input, state, Boolean decisions, and testing to a first complete while-loop program.
- Evidence habit: trace first, then construct one connected program and defend tests.
- Independence level: Planning frame supplied; the complete elevator loop is student-authored.

## How to work

1. Complete the trace and explain the state evidence.
2. Diagnose the supplied defect and defend a repair.
3. Write one connected program that satisfies the complete specification.
4. Record tests, expected results, and a brief defense.

Python operators `and`, `or`, and `not` remain lowercase and appear as bold dark-teal tokens so they are distinct from ordinary prose.
