This page explains what is included in the current student learning collection.

## Included

The current collection includes student-facing learning materials that students can use for preparation, class work, review, synthesis, or transition planning:

- daily workbooklets;
- Week 1 checkpoint/practice;
- Day 19 support/transition packet;
- student guidance, calendar, journey, quick glossary, and competency navigation files.

Day 20 remains visible only as a ceremony/no-normal-bridge-class boundary. No Day 20 student packet is included; use Day 19 or explicit instructor/LMS directions for final support or reflection.

## Excluded from this repository view

Daily quiz PDFs and exam/assessment PDFs are excluded from the current collection and from the repository history prepared for public or organization publication.

Instructor-only materials, answers, keys, source files, build logs, and grading notes are not part of the student collection.

## Notebook practice

Approved notebook practice remains in the companion notebook repository. This repository links to that path instead of copying notebooks here, so students can find practice without mixing notebook maintenance files into the PDF collection.
