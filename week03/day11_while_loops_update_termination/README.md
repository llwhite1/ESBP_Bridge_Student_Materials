## What you are learning

- Target: Design loops whose state changes make termination demonstrable.
- Evidence habit: condition result, state update, sentinel handling, and termination evidence.
- Independence level: Termination evidence prompted; the algorithm is no longer sequenced line by line.

## How to work

1. Complete the trace and explain the state evidence.
2. Diagnose the supplied defect and defend a repair.
3. Write one connected program that satisfies the complete specification.
4. Record tests, expected results, and a brief defense.
